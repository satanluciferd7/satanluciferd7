# 🇳🇵 Nepal Portal

A comprehensive civic portal for Nepal featuring government services, community engagement, and cybersecurity awareness.

## Features

- **Government Services**: Interactive guides for citizenship, passport, driving license, and more
- **Community Platform**: News, problem reporting, and discussion forums
- **Cybersecurity Tools**: Password strength checker and security tips
- **Mobile Support**: PWA with offline capabilities and dark mode
- **Admin Dashboard**: Content moderation and user management

## Quick Start

1. Open `index.html` in your browser
2. Register an account or login
3. Explore government service guides
4. Report community problems
5. Access admin features (admin users only)
